# Learn By Hariom

Learn By Hariom is a modern, production-ready full-stack learning platform built with React, Vite, TypeScript, Tailwind CSS, Express, MongoDB, and Mongoose.

## Features
- Responsive React frontend with dark/light theme
- SEO-friendly pages and PWA support
- Express API with authentication, CRUD endpoints, and Swagger docs
- MongoDB-backed content models for projects, courses, blogs, roadmaps, notes, gallery, certificates, contacts, newsletters, and settings
- Docker, GitHub Actions, linting, formatting, and testing support

## Getting Started

### Prerequisites
- Node.js 20+
- MongoDB

### Install
```bash
npm install
```

### Run locally
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Docker
```bash
docker compose up --build
```
